# 🔌 Obsidian Plugin Tavsiyalari

## Majburiy Pluginlar

### 1. Templater
- **Maqsad:** Avtomatik template'lar
- **O'rnatish:** Community Plugins → Templater
- **Sozlash:**
  - Template folder: `Templates`
  - Trigger on file creation: ✅
  - Hotkey: `Ctrl+P` → "Templater: Insert template"

### 2. Dataview
- **Maqsad:** Dinamik query'lar va jadvallar
- **O'rnatish:** Community Plugins → Dataview
- **Sozlash:**
  - Enable JavaScript queries: ✅
  - Inline queries: ✅

### 3. Calendar
- **Maqsad:** Kunlik yozuv navigatsiyasi
- **O'rnatish:** Community Plugins → Calendar
- **Sozlash:**
  - Default daily note folder: `Daily`
  - Week start: Monday

### 4. Kanban
- **Maqsad:** Loyiha boshqaruvi
- **O'rnatish:** Community Plugins → Kanban
- **Foydalanish:** `Ctrl+P` → "Create new Kanban board"

### 5. Spaced Repetition
- **Maqsad:** Flashcard takrorlash
- **O'rnatish:** Community Plugins → Spaced Repetition
- **Sozlash:**
  - Flashcard tag: `#flashcard`
  - Daily review limit: 20
  - New cards per day: 10

## Foydali Pluginlar

### 6. Tasks
- **Maqsad:** Task tracking
- **O'rnatish:** Community Plugins → Tasks
- **Foydalanish:** `Ctrl+P` → "Tasks: Create or edit task"

### 7. QuickAdd
- **Maqsad:** Tezkor note qo'shish
- **O'rnatish:** Community Plugins → QuickAdd
- **Sozlash:** Capture → `00-Inbox` papkasiga

### 8. Excalidraw
- **Maqsad:** Diagramma/chizma
- **O'rnatish:** Community Plugins → Excalidraw

### 9. Various Complements
- **Maqsad:** Auto-complete
- **O'rnatish:** Community Plugins → Various Complements

### 10. Periodic Notes
- **Maqsad:** Haftalik/Oylik yozuvlar
- **O'rnatish:** Community Plugins → Periodic Notes
- **Sozlash:**
  - Weekly note folder: `Reviews`
  - Weekly template: `Templates/Weekly Review`

### Templates
- Template folder: `Templates`

### File Recovery
- Auto-save: 1 minut
- Backup: ✅
